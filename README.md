## Urban Zest Restaurant

### Project Overview

**Urban Zest Restaurant** is a modern, responsive restaurant website built using **HTML**, **CSS**, and **Bootstrap 5**.  
It showcases an elegant and user-friendly design that highlights the restaurant’s menu, story, and contact options.

The project demonstrates understanding of **layout structuring**, **Bootstrap grid systems**, **media responsiveness**, and **component styling**.

### Features

- **Responsive Navbar:**  
  A Bootstrap navigation bar with brand name and links that collapse into a mobile menu.
- **Hero Section:**  
  A large banner with a background image, centered heading, and call-to-action button.
- **Menu Section:**  
  Six interactive Bootstrap cards showcasing signature dishes with descriptions and prices.
- **About Section:**  
  Split layout with an image and text explaining the restaurant’s story.
- **Contact Section:**  
  A form for user messages and a complementary image for design balance.
- **Footer:**  
  Includes copyright, social icons (Font Awesome), and a design credit.

---

### Tools & Technologies

- **HTML5**
- **CSS3**
- **Bootstrap 5**
- **Font Awesome Icons**

## Horizen Travel Co.

### Project Overview

**Horizen Travel Co.** is a visually appealing and responsive travel agency web app designed to help users explore destinations and book tour packages.  
It highlights creativity in using **Bootstrap utilities** to build a structured layout with rich visuals and clean typography.

### Features

- **Navigation Bar:**  
  A clean Bootstrap navbar with links to Home, Destinations, Tours, and Contact.
- **Hero Section:**  
  A full-width hero with a scenic background image, heading, and subheading centered using Bootstrap utilities.
- **Destinations Section:**  
  Uses Bootstrap’s responsive grid to showcase destinations with titles and short descriptions.
- **Tour Packages Section:**  
  Displays six curated tour packages, each with a title and short description.
- **Contact Section:**  
  Simple and clear contact area for user engagement.
- **Fully Responsive Layout:**  
  Adjusts seamlessly across mobile, tablet, and desktop screens.

### Tools & Technologies

- **HTML5**
- **CSS3**
- **Bootstrap 5**
- **Font Awesome**
